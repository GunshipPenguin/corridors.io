## MazeGame
#### By GunshipPenguin (Rhys RE)

Generates a simple maze using a depth first search algorithm and turns it into a 3d maze using three.js. The user is placed in the upper left hand corner of the maze and must try to find his/her way to the bottom right hand corner of the maze, which is marked by a hole in the wall.

Controls

* WASD - Move
* Mouse - Adjust yaw / look
